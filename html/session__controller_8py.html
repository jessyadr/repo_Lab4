Description détaillée

Gestion des endpoints REST pour les séances.

Ce fichier contient les fonctions nécessaires pour manipuler les séances associées aux modules d'un cours via l'API.

Dépendances/Modules

  • connexion : Framework pour gérer les APIs REST.
  • json : Module standard pour manipuler les fichiers JSON.
  • os : Utilisé pour vérifier l'existence du fichier de données.
