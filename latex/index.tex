\hypertarget{index_description_main}{}\doxysection{\texorpdfstring{Description}{Description}}\label{index_description_main}
Ce projet implémente une API REST basée sur Swagger pour gérer le contenu des cours. Il utilise le framework Connexion pour exposer des endpoints REST. Ce fichier principal configure et lance le serveur de l\textquotesingle{}API.\hypertarget{index_notes_main}{}\doxysection{\texorpdfstring{Notes}{Notes}}\label{index_notes_main}

\begin{DoxyItemize}
\item Le serveur écoute par défaut sur le port 8080.
\item Assurez-\/vous que le fichier {\ttfamily swagger.\+yaml} est correctement configuré.
